# Caesar Cipher Encryption and Decryption

## Objective
Create a program that encrypts and decrypts text using the Caesar Cipher algorithm.

## Description
This project allows users to:
- Enter a message
- Enter a shift value
- Encrypt the message
- Decrypt the message

## Algorithm
1. Accept text input
2. Accept shift value
3. Shift each alphabet character by the given value
4. Display encrypted/decrypted output

## Features
✔ Encrypt text  
✔ Decrypt text  
✔ Supports uppercase and lowercase letters  

## Technologies Used
- Python
